What is Teamruns?

Teamruns is a both online and couch co-op multiplayer mod for OpenGOAL, the PC port Jak and Daxter.

The mod allows you to play through the game with others in a shared game world with interactable players. All ingame cutscenes that normally interrupt the game flow have also been modified to run in real time to work better with multiplayer.


The mod in general is mostly catered towards speedrunning with built in ingame timer, splits, and automatic leaderboard submission and verification through the use of demo files, but does also come with some other game modes for casual play and a recording tool for creating custom demo files.

Features still missing from the multiplayer aspect are navigation and projectile pathing logic for normal enemies that are still individual to each player.
