Getting Started Guide

1.Download the latest release from above.

2.Install/Run the client.

2.5.IF you intend to do speedruns, create a user in the client.

3.Provide your ISO of Jak & Daxter to the client when prompted.

4.Read the "Help/Info" page found in the lobby page.

5.Create or join a game/lobby.

6.Launch the game from the side menu in the top left.

What is Teamruns?

Teamruns is both a online and couch co-op multiplayer mod for the pc port of Jak and Daxter OpenGOAL, for more info check the FAQ page.
