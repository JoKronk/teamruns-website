Frequently Asked Questions

Game Questions

Is this online multiplayer? #

Yes.

Does this support couch co-op? #

Yes both couch co-op and online are supported, both individually and at the same time.

What is changed compared to the normal game? #

All in game cutscenes play in real time without iterrupting gameplay with cutscenes.
Bosses are synced by phases, if one player dies while someone else in the fight it will be set back one phase.

For Speedruns ONLY: Players need to meetup at the zoomer between each hub world before they will go at full speed.

What is synced/shared between players and what isn't? #

Everything except enemies and projectile flight paths should be synced.

Are you planning on adding the remaining things missing? #

Hopefully yes, I have some plans for how to do these but I also want to take a break after the initial full release.

Can I disable the extra stuff on the screen? #

Yes, hold down L2 and R2 then use left and right on the dpad to select UI element and up and down to toggle them.

Issues

I am unable to see other players in the game, help! #

There's sometime some issues with this that I'm working on, try some of the following:

- Check that none of you are a spectator in the lobby.

- Relaunch the game.

- Rejoin the lobby.

- Recreate the lobby.

- Relaunch the client.

Double NAT issue. #

Since the client operates on peer to peer connections it might have trouble establishing one if you're behind a double NAT.
If Teamruns doesn't seem to work for you at all in terms of connecting with others this might be the issue but I've only had one instance of a person with this issue so far, I'll hopefully be looking into ways to get around it somewhat soon though (setting up a TURN server).

The game crashed! #

Check if you can replicate it and contact me on discord.

Project Info

Why is this not on the Mod Launcher? #

The Mod Launcher has been mainly written to handle mods that consists soley of the game itself, as this requires a separate client to handle the networking it's a bit of an exception.

Are you planning on doing Jak II / 3 / X? #

Not sure, I originally wanted to before I fully understood the scope of the whole project.
Either way the missing systems in Jak 1 would be benefitial to have in place before starting anything on the other games.

Can I help with the project? #

If you want to, this by no means needs to be a solo project, contact me on discord.

Are you planning on createing more game modes / mods for Teamruns? #

I hope so but fixing the remaining systems and adding better mod support in general should come first.

Can I create a multiplayer mod for/from Teamruns? #

Support for this is quite bad at the moment but is something I hope to improve.

How are updates handled? #

New updates are automatically given to the Client in a update prompt for both the OpenGOAL and Client updates.

User Creation

Do I need to create a user? #

No, a user is only needed if you plan to do speedruns and want your time on the leaderboard.

How is my password stored if I create a user? #

Users including passwords are handled by Firebase, I personally do not have access to see how or where the passwords are stored, the code for the client is also open source and can be audited at any time.
